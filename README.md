# Path Finding Visualizer

An interactive web-based Path Finding Visualizer built using HTML, CSS, and JavaScript.

## Features

- Interactive 20x20 grid
- Start and End node selection
- Wall/obstacle creation
- BFS Algorithm
- DFS Algorithm
- Dijkstra Algorithm
- A* Algorithm
- Real-time visualization
- Shortest path highlighting
- Speed control slider
- Reset and Clear Walls buttons
- Dark theme UI

## Technologies Used

- HTML
- CSS
- JavaScript
- Data Structures and Algorithms

## How to Run

1. Clone the repository.
2. Open index.html in a browser.
3. Select an algorithm.
4. Place Start, End, and Walls.
5. Click Start to visualize the algorithm.
